Documentation: subsidized housing owners (NHPD) columns

  • nhpd_property_id, property_name, property_address, city, state, zip, census_tract — property identity and location.
  • property_status, active_subsidies, total_inactive_subsidies, total_units — current subsidy status and unit count.
  • owner, owner_type — the real legal owner name and type (For Profit / Non-Profit / Public Entity / Limited Dividend / Multiple).
  • manager_name, manager_type — the property management company, which may differ from the owner.
  • earliest_start_date, latest_end_date — the subsidy program's date range.

Documentation: apartment parcel ownership (MOD-IV) columns

  • prop_loc, mun_name, block, lot, census_tract — the parcel's location and tax identifiers.
  • st_address, city_state, zip_code, owner_mailing_key — the owner's mailing address as recorded by NJ's tax assessor. Owner name is not available in this extract — see the methodology page.
  • land_val, imprvt_val, net_value — assessed property values.
  • sale_price, deed_date — the most recent recorded sale, if any.
  • yr_constr, dwell — year built and number of dwelling units.
  • out_of_state_owner — whether the mailing address is outside New Jersey.
